# AgroSense - API & Backend ⚙️

Bem-vindo ao repositório oficial do **Squad API** do projeto AgroSense.
Aqui ficam as regras de negócio, banco de dados e endpoints.

## ⚠️ Regra de Ouro (LEIA ANTES DE CODAR)
A branch `main` está **bloqueada** para commits diretos.

**Como trabalhar:**
1. Sempre crie uma nova branch para sua tarefa: `git checkout -b feature/criar-models`
2. Faça seus commits.
3. Abra um **Pull Request (PR)** para a `main` quando terminar.

---
**Stack:** Python, Django Rest Framework (DRF), PostgreSQL.
